## 🚀 Instalasi & Penggunaan

1. **Clone repository:**

   ```sh
   git clone https://github.com/username/My_project.git
   cd My_project
   ```

2. **Install dependencies:**

   ```sh
   npm install
   ```

3. **Jalankan Tailwind CLI (development):**

   ```sh
   npm run tw
   ```

   atau build production:

   ```sh
   npm run build
   ```

4. **Buka `index.html` di browser.**
